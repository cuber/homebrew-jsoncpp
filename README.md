homebrew-jsoncpp
================

Thanks to [Json-cpp](https://github.com/mrtazz/json-cpp "Json Parser for C/C++")

Usage.
-----------------
``` bash
brew tap cuber/homebrew-jsoncpp
brew install jsoncpp
```

Code.
-----------------
``` c++
# include header
include <json/json.h>
```
``` makefile
# compile with option
-ljson
```
